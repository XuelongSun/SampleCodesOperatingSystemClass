#### 系统集成
+ 将CPU模拟、进程管理、内存管理、文件系统等模块有机整合

+ 实现交互式命令行界面，命令提示符体现自己的特点，如张三同学的系统提示符为`ZS-OS>`:
    ```cmd
    ZS-OS> sysinfo          # 显示系统信息 - Lab1:这个系统属于命令行接口
    ZS-OS> run [program]    # 运行程序 - Lab3/4
    ZS-OS> meminfo          # 显示内存状态 - Lab5/6
    ZS-OS> ls [path]        # 列出文件 - Lab7
    ZS-OS> exp producer-consumer # 进程同步实验演示 - Lab2
    ZS-OS> exp memory-allocation # 实验演示 - Lab5
    ```
